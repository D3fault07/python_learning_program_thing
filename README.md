| **Function**          | **Description**                                                                 |
|-----------------------|---------------------------------------------------------------------------------|
| `clear_text()`        | Clears the content of a text widget.                                            |
| `display_text()`      | Displays provided content in a text widget.                                     |

| **Function**          | **Description**                                                                 |
|-----------------------|---------------------------------------------------------------------------------|
